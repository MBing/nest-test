# NestJS Example

## Description

This example is made to show a Postgres setup with NestJS. 
It creates tasks and has no other use than to be an example.

## Tools Needed
- Postgres [Postgresql.org](https://www.postgresql.org/)
- A Postgres Client [PgAdmin Mac](https://www.pgadmin.org/download/pgadmin-4-macos/)
- Postico []()

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
